# Unity 2.3 Code

This folder contains the code-focused portion of the Unity project from `E:\2.3`.

## Included

- `Assets/Scripts`: gameplay, UI, camera, platform, bubble, teleport, respawn and level scripts
- `Assets/Editor`: Unity editor tooling scripts
- `Assets/InputSystem_Actions.inputactions`: input action configuration
- `Packages`: Unity package manifest and lock file
- `ProjectSettings/ProjectVersion.txt`: Unity editor version

## Excluded

The following files were intentionally not uploaded because they are generated, local-only, or too large for a code showcase:

- `Library/`
- `Logs/`
- `UserSettings/`
- build outputs and temporary files
- `3.0.zip`
- large art/audio/font assets

## Resume Description

Unity 2D game codebase with custom C# gameplay systems for player movement, bubble mechanics, teleportation, camera effects, level flow, UI menus, localization, and editor tooling.
